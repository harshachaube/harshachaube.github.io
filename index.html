01 ·

Azure AI Pipeline · Retail Procure-to-Pay

Designed and delivered a full Azure AI solution for a leading UK retail client across their procure-to-pay cycle. Raw invoice and PO documents land in Azure Blob Storage containers before flowing through a pipeline via Azure Data Factory. A Document Intelligence agent handles automated extraction from unstructured invoices — with a human-in-the-loop review layer for low-confidence outputs before they're committed downstream. Structured P2P data is served via Azure Synapse Analytics for SQL-based querying and reporting, while code agents within AI Foundry orchestrate anomaly detection and exception flagging across the pipeline — surfacing high-risk transactions for analyst review without manual triage.
